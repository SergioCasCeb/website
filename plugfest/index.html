Services to support the W3C WoT Plugfests

We will provide sevices to support the operation of a Plugfest of W3C WoT

TD Directory

REST interface to add, update and query TDs for discovery purposes. See how to interact with the directory here (Swagger specification).

  • HTTP TD Repository (Inactive)
  • CoAP TD Repository (Inactive)

TD Playground

Tool to check the validity of a TD against its definition in the Current Practices document. It performs both syntactic checks and semantic checks (see ontology & its core principles).

  • Playground

WebUI

  • Browsified node-wot, see http://plugfest.thingweb.io/webui/
  • Browser Examples
    • Smart Coffee Machine
    • Counter
  • JavaFX-based UI (see WoT-FXUI)
    • See running Web-UI instance in browser without any plugin
    • Possible as a local desktop application and an Android/iOS app also
